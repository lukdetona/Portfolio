Projetos

jokempo

este código cria uma página de jogo interativa de "Pedra, Papel e Tesoura". O usuário faz sua escolha clicando em uma das imagens, o computador escolhe aleatoriamente, e o resultado do jogo é exibido na página. É um exemplo simples de como HTML, CSS e JavaScript podem ser combinados para criar uma experiência interativa na web.

cardapio

este código oferece aos usuários a capacidade de criar um cardápio semanal, adicionar, editar e excluir itens alimentares, bem como monitorar sua alimentação ao longo dos dias da semana. É uma aplicação simples de gerenciamento de cardápio, usando HTML, CSS e JavaScript para interatividade.

detran

este código cria uma calculadora de excesso de velocidade que fornece informações sobre penalidades com base na diferença entre a velocidade do carro e o limite de velocidade permitido. Ele também permite ao usuário inserir o nome do condutor e o número da habilitação para incluí-los na mensagem de resultado.
